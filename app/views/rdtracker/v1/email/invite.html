Multiple choice - one response (grid) View item design

Item design overview

Aspect ratio: 4:3 (800px by 600px safe area)

Container size: 200px by 200px with 20px inner padding

Margin: 64px between each component

Interaction type: Touch or select

Digital feedback: Yellow focus indicator (highlight)

A wireframe of the multi choice grid item, demonstrating the 4:3 aspect ratio, margins and paddings around the 4 different user interface elements.

A mockup of the multi choice grid item.

What's working for users?

  • Most users instantly recoginse that the intended interaction for this item is to touch an image, and can do so with little to no usability issues

What's isn't working for users?

  • Some users think that pointing is the appropriate response to this question, but are able to interact once prompted by a practitioner
  • In some cases, we see users with too heavy/light touches that can lead to a light touch response not registering, or a heavy touch causing the screen to move.

Next steps

  • We are in the process of exploring a 'familiarisation' area, where users can practice their touch responses ahead of an assessment. We will conduct research to determine whether users interact with this item more intuitively once they have seen a familiarisation area.

Design justifications

Visual feedback

Every action a user does it should provide a visible, understandable and immediate reaction. Feedback provides a sense of comfort, control and security, it tells our users if they're doing the right thing.

Forgiving design

Using large touch areas and generous spacing between each user interface element takes into account our users innacuracy and developing motor skills.

Intelligent consistency

Encouraging the use of similar screens for similar functions, this means we can duplicate and modify new designs quickly but more importantly, knowing that the layout and user interface components used will work through evidenced research findings.

Container size

The container in which the images are placed in are 200px by 200px (this dimension is generally used across majority of the item types, specific item types require different sized containers), this allows for sufficient space to place an illustration which can be sized to an easily recognisable or understandable scale.

Responsive design

By containing the overall design within a 4:3 aspect ratio, the designs will adapt and respond to different screen resolutions and devices.
